Avoid public visibility in unit-tests

Noncompliant Code Example

@RunWith (MockitoJUnitRunner.class)
class AvoidPublicVisibilityInUnitTest {
    public String SOME_CONST = "some value";
}

Compliant Solution

@RunWith (MockitoJUnitRunner.class)
class AvoidPublicVisibilityInUnitTest {
    private String SOME_CONST = "some value";
}
